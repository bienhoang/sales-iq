# Content Repurpose Matrix

## Blog Post Can Become

| Target Format | Target Platform | Key Adaptation | Effort |
|:-------------|:----------------|:---------------|:-------|
| Thread (5-10 tweets) | X/Twitter | Extract key points, make each tweet standalone | Quick |
| Carousel (5-10 slides) | Instagram | Visual-first, one insight per slide, hook on slide 1 | Medium |
| Long-form post | LinkedIn | Add personal angle, professional framing | Quick |
| Short video script | TikTok | Hook in 3 sec, pick 1 key insight, CTA | Medium |
| Newsletter feature | Email | Summarize + link to full post, add personal note | Quick |
| Discussion prompt | Discord/Slack | Ask a question from the blog's main thesis | Quick |
| Infographic brief | Design team | Key data points and flow for visual treatment | Heavy |
| Podcast talking points | Audio | Expand conversationally on the topic | Medium |
| Comparison page data | Blog | Add case study data to vs. pages | Medium |

---

## Webinar / Video Can Become

| Target Format | Target Platform | Key Adaptation | Effort |
|:-------------|:----------------|:---------------|:-------|
| Blog post summary | Blog | Transcribe key sections, add structure and SEO | Heavy |
| Quote graphics (3-5) | Instagram/LinkedIn | Best sound bites as designed quote cards | Medium |
| Short clips (15-60s) | TikTok/Reels | Timestamp the best moments, add captions | Medium |
| Email recap | Email | Summary + recording link + key takeaways | Quick |
| Social highlights thread | X/Twitter | "5 things I learned from..." format | Quick |
| Community discussion | Discord/Slack | "We discussed X — what do you think?" | Quick |

## Customer Case Study Can Become

| Target Format | Target Platform | Key Adaptation | Effort |
|:-------------|:----------------|:---------------|:-------|
| Social proof snippets | All social | Pull the best metric or quote | Quick |
| Email testimonial block | Email | Short quote + result + CTA | Quick |
| Ad copy | Meta/Google/LinkedIn | Results-focused headline + social proof | Quick |
| Community success story | Discord/Slack | Celebrate the customer, tag if possible | Quick |
| Comparison page data | Blog | Add to "[Product] vs [Competitor]" pages | Medium |
| Sales deck slide | Internal | Metric + logo + one-liner | Quick |

## Product Changelog Can Become

| Target Format | Target Platform | Key Adaptation | Effort |
|:-------------|:----------------|:---------------|:-------|
| Feature announcement email | Email | Benefit-led, not feature-led | Medium |
| Social posts | All social | "Just shipped" excitement + screenshot | Quick |
| Community update | Discord/Slack | Casual, invite feedback | Quick |
| Blog tutorial | Blog | "How to use [new feature]" SEO post | Heavy |

## Repurpose Priority Guide

**Quick wins (do these first)**:
- Blog → X thread (15 min, high reach potential)
- Blog → LinkedIn post (15 min, professional audience)
- Case study → social proof snippet (10 min, all channels)
- Email → LinkedIn post (20 min, expands reach)

**Medium effort (schedule weekly)**:
- Blog → Instagram carousel (requires design)
- Webinar → email recap (requires editing)
- Blog → TikTok script (requires filming)

**Heavy lift (plan quarterly)**:
- Webinar → full blog post (requires transcription + SEO)
- Case study → pillar page section (requires writer)
- Infographic brief → designed asset (requires designer)
