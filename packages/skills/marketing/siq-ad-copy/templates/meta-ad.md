## Single Image / Video Ad

### Primary Text (125 chars above fold, 500 max)
```
Version A (pain-led):
[Hook with the pain point. First 125 chars must stop the scroll. Then expand with benefit + proof + CTA.]

Version B (benefit-led):
[Open with the outcome. "What if you could [desired result] without [obstacle]?" Then support with how.]

Version C (social proof-led):
["[X] teams already [achieved result] with [product]." Then explain why and invite them in.]
```

## Retargeting Ad Variations

For users who visited the site but didn't convert:

```
Version A — Objection handler:
"Still thinking about it? Here's what [X] customers said after their first week: [quote or stat]."

Version B — Offer:
"Ready to start? Get [X days] free — no credit card required."

Version C — FOMO:
"[X] teams signed up this week. Here's what they're already doing with [product]."
```

## Visual Specs

| Format | Recommended Size | Aspect Ratio | Max File Size |
|:-------|:----------------|:-------------|:--------------|
| Feed (image) | 1080 x 1080px | 1:1 | 30MB |
| Feed (image landscape) | 1200 x 628px | 1.91:1 | 30MB |
| Story / Reel | 1080 x 1920px | 9:16 | 30MB (image) / 4GB (video) |
| Carousel card | 1080 x 1080px | 1:1 | 30MB per card |

**Text overlay rule**: Keep text under 20% of image area for best delivery.

## Targeting Setup

| Dimension | Recommendation |
|:----------|:--------------|
| **Core audience** | Age, location, interests matching ICP |
| **Custom audience** | Website visitors (180 days), email list, video viewers |
| **Lookalike** | 1-2% lookalike of best customers |
| **Exclusions** | Current customers, recent converters |
| **Placements** | Start with Advantage+ placements, then narrow by performance |
