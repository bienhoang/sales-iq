# Onboarding Drip Template (10 emails, 15 days)

**Trigger**: New trial or paid account created
**Goal**: Guide users to activation milestones → reduce time-to-value → convert trial → retain paid
**Key principle**: Milestone-based. If a user completes the action before the email sends, skip it.

---

## Milestone Map

```
Account Created
     ↓
[M1] Profile/setup complete       ← Email 2 drives this
     ↓
[M2] First core action taken      ← Email 3 drives this
     ↓
[M3] Key feature explored         ← Email 4 drives this
     ↓
[M4] Integration connected        ← Email 5 drives this
     ↓
[M5] Advanced feature used        ← Email 6 drives this
     ↓
[Activation] — User sees real value → upgrade or retain
```

---

## Email 1 — Getting Started (Day 0)

**Subject**: "Your [Product] setup guide (takes 10 minutes)"
**Preview**: "3 steps to your first result."

```
Hey {{first_name}},

Welcome! Let's make the next 10 minutes count.

To get the most out of [Product], I recommend this order:

✓ Step 1: [Setup action — e.g., "Set up your brand profile"] → [link]
✓ Step 2: [Core action — e.g., "Run your first skill"] → [link]
✓ Step 3: [Integration — e.g., "Connect your workspace"] → [link]

Each step builds on the last. Start with Step 1 now — it takes about 3 minutes.

[CTA: "Start setup →"]

[Name]
```
