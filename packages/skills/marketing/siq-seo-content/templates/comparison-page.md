# Comparison Page Template

Format: "[Your Product] vs [Competitor]"

**Purpose**: Capture high-intent BOFU traffic from buyers actively evaluating options.
**Tone**: Fair and honest — biased comparison pages backfire. Let your strengths speak for themselves.
**Target length**: 1,500–2,500 words

---

## Content Brief

| Field | Value |
|:------|:------|
| **Title** | "[Your Product] vs [Competitor]: Which Is Right for You? ([Year])" |
| **Target Persona** | [ICP — buyer in evaluation stage] |
| **Funnel Stage** | BOFU |
| **Primary Keyword** | "[your product] vs [competitor]" |
| **Secondary Keywords** | "[competitor] alternative", "best [category]", "[competitor] pricing" |
| **Content Goal** | Conversions |
| **CTA** | Start free trial / Book demo |

---

## SEO Metadata

```
Title Tag: [Product] vs [Competitor]: [Year] Comparison
Meta Description: Honest [Product] vs [Competitor] comparison. See features, pricing, and which is best for [use case]. Updated [Month Year].
URL Slug: /compare/[your-product]-vs-[competitor]
```
