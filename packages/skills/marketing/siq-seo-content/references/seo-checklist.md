# SEO Content Checklist

Use this before publishing any SEO content piece.

---

## On-Page SEO

### Title & Meta
- [ ] Title tag is under 60 characters
- [ ] Title tag contains the primary keyword (preferably near the start)
- [ ] Title tag is compelling and click-worthy (not just keyword-stuffed)
- [ ] Meta description is under 160 characters
- [ ] Meta description contains the primary keyword naturally
- [ ] Meta description has a clear value proposition or hook
- [ ] URL slug is keyword-rich, short, and hyphenated (no underscores)

### Content Structure
- [ ] H1 contains the primary keyword (only one H1 per page)
- [ ] H2s and H3s follow logical hierarchy
- [ ] Primary keyword appears in first 100 words of body text
- [ ] Keyword density is 0.5-1.5% (not stuffed, not absent)
- [ ] At least one H2 contains the primary keyword or a close variant
- [ ] Subheadings appear every 200-300 words for scannability
- [ ] Paragraphs are 2-4 sentences max (no walls of text)
- [ ] Content is the target word count (blog: 1,500-2,500; pillar: 3,000-5,000)

### Links
- [ ] 3-5 internal links to relevant pages on your site
- [ ] 2-3 external links to authoritative, non-competing sources
- [ ] All internal links use descriptive anchor text (not "click here")
- [ ] No broken links

### Images & Media
- [ ] All images have descriptive alt text
- [ ] Primary keyword appears in at least one alt text (naturally)
- [ ] Images are compressed for web (under 200KB where possible)
- [ ] At least one image included (improves time-on-page)

### Featured Snippet Optimization
- [ ] FAQ section included (3-5 questions)
- [ ] FAQ answers are concise (40-60 words — ideal for featured snippets)
- [ ] Direct answer to main query appears near top of article
- [ ] Tables, lists, and step-by-step formats used where appropriate

---

## Content Quality

### Substance
- [ ] Content genuinely answers the search query better than current top results
- [ ] Original research, insights, or examples included (not just rephrased content)
- [ ] Claims backed by data, sources, or examples
- [ ] Practical takeaways — reader can act on this content
- [ ] "Would I share this?" test passed

### Audience Alignment
- [ ] Written for the target ICP (from icp-profiles.md)
- [ ] Matches the correct funnel stage (TOFU/MOFU/BOFU)
- [ ] Brand voice consistent with brand-voice-guide.md
- [ ] CTA is relevant to funnel stage (don't ask for purchase at TOFU)

### Freshness
- [ ] Publish date included
- [ ] Content is accurate and up to date
- [ ] Stats and data cited with sources and dates
- [ ] Set a reminder to update this content in 6-12 months

---

## Technical SEO (Confirm with Dev Team)

- [ ] Page loads under 3 seconds (check PageSpeed Insights)
- [ ] Mobile-responsive layout
- [ ] Canonical tag set correctly (no duplicate content issues)
- [ ] Schema markup added if applicable (Article, FAQ, HowTo)
- [ ] Open Graph tags set (title, description, image for social sharing)
- [ ] No indexing issues (page is not accidentally noindexed)

---

## Post-Publish

- [ ] Submitted to Google Search Console for indexing
- [ ] Internal links from related existing pages added pointing to this new page
- [ ] Social distribution planned (which channels, which angle)
- [ ] Added to content calendar for future update review
- [ ] Tracked in keyword ranking spreadsheet
