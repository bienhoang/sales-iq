# Community Escalation Playbook

## When to Escalate

Not every issue needs the same response. Use this guide to calibrate.

| Severity | Examples | Response Time | Who Handles |
|:---------|:---------|:-------------|:------------|
| **Low** | Off-topic post, minor formatting issue | 24 hrs | Any moderator |
| **Medium** | Self-promotion without disclosure, repeated low-effort posts | 4 hrs | Moderator |
| **High** | Harassment, heated conflict, spam wave | 1 hr | Community Lead |
| **Critical** | Hate speech, doxxing, illegal content, scam | Immediately | Community Lead + escalate to leadership |

## Response Templates

### Low: Off-Topic Post

**In-thread (friendly redirect)**:
```
Hey [Name] ‚Äî great question! This might get more traction in #[relevant-channel].
Mind if I move it there? [Or: I'll leave a pointer here for others.]
```

**DM (if pattern)**:
```

```

### Medium: Heated Discussion

**In-thread (de-escalate)**:
```

**DM to both parties**:
```

### High: Harassment

**Immediate in-thread action**:
```
[Remove the message]
[Post in thread]: A message has been removed for violating our community guidelines.
If you were affected, please DM me directly.
```

**DM to target**:
```
Hi [Name], I saw what happened in #[channel] and I've removed the message.
I'm really sorry you experienced that here.
Are you okay? Is there anything you need from me right now?
[Moderator name]
```

**DM to offender**:
```
Hi [Name], your message in #[channel] was removed because it violated our guidelines around [harassment/hate speech/etc.].
This is a serious violation.
[First offense]: This is a formal warning. Further violations will result in removal from the community.
[Repeat offense]: Your account has been [muted for X days / removed from the community].
If you believe this was an error, reply here.
[Community Lead name]
```

---

### Critical: Illegal Content / Scam / Hate Speech

**Immediate actions (in this order)**:
1. Screenshot / document the content before removing
2. Remove the content immediately
3. Ban or suspend the account
4. Notify Community Lead within 15 minutes
5. Document in the incident log (date, content summary, action taken)
6. If content is potentially illegal (CSAM, credible threats): escalate to leadership + legal immediately

**Do NOT**:
- Engage in extended conversation with the offender
- Share the content with others (even as warning)
- Delay action to get more information ‚Äî act first, document second

## Support Escalation (Product Issues)

When a member has a product problem that community can't solve:

**Response in community**:
```
Hey [Name], this sounds like something our support team can help with directly.
‚Üí [Support email / help center link]
They're usually pretty quick ‚Äî I'd expect a response within [X] hours.
Let us know here if you're still stuck after reaching out!
```

**Internal handoff** (if you have a support Slack):
```
@support-team ‚Äî [Name] in [Discord/Slack community] is experiencing [brief description].
I've pointed them to the help desk but flagging here in case it needs priority attention.
[Link to their community message]
```
