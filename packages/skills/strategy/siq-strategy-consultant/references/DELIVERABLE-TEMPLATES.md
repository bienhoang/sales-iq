# Deliverable Templates — siq-strategy-consultant

9 structured output templates for common consulting deliverables. Use these to ensure
consistent, professional, executive-ready output.

---

## T1: Executive Summary (SCQA Format)

```
# Executive Summary: [Topic]

## Situation
[1–2 sentences: current state and context. What is true today?]

## Complication
[1–2 sentences: what has changed or what problem has emerged?]

## Question
[The single key question this analysis answers]

## Answer
[2–3 sentences: the core recommendation and expected impact]

---

## Key Recommendations
1. [Action 1] — [Expected impact] — [Owner] — [Timeline]
2. [Action 2] — [Expected impact] — [Owner] — [Timeline]
3. [Action 3] — [Expected impact] — [Owner] — [Timeline]

## Supporting Evidence
- [Key finding 1 — fact or validated assumption]
- [Key finding 2 — fact or validated assumption]
- [Key finding 3 — fact or validated assumption]

## Assumptions (label clearly)
- [Assumption 1]
- [Assumption 2]
```

## T6: OKR Framework

```
# OKRs: [Team / Company] — [Quarter / Year]

## Objective 1: [Ambitious, qualitative goal — inspiring, not a metric]
*Why it matters: [1 sentence rationale]*

| Key Result | Target | Current | Status |
|---|---|---|---|
| KR 1.1: [Measurable, binary outcome] | [Number] | [Number] | On Track / At Risk / Off Track |
| KR 1.2: [Measurable outcome] | [Number] | [Number] | On Track / At Risk / Off Track |
| KR 1.3: [Measurable outcome] | [Number] | [Number] | On Track / At Risk / Off Track |

Key Initiatives:
- [Initiative 1] → drives KR 1.1
- [Initiative 2] → drives KR 1.2

---

## Objective 2: [Ambitious, qualitative goal]
*Why it matters: [1 sentence rationale]*

| Key Result | Target | Current | Status |
|---|---|---|---|
| KR 2.1: [Measurable outcome] | [Number] | [Number] | On Track / At Risk / Off Track |
| KR 2.2: [Measurable outcome] | [Number] | [Number] | On Track / At Risk / Off Track |

Key Initiatives:
- [Initiative 3] → drives KR 2.1

---

## OKR Health Check
- Objectives are inspiring and directional (not a KPI themselves)
- Each KR is measurable, binary, and owner-assigned
- No more than 3 objectives and 3–5 KRs each
- Initiatives clearly map to at least one KR
```
